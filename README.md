# DevSecOps CI CD Pipeline Simulation

This example demonstrates a simplified DevSecOps CI/CD pipeline, emphasizing the integration of security checks early in the development lifecycle. It simulates static application security testing (SAST) and software composition analysis (SCA) as 'shift-left' security gates. The pipeline will intentionally fail early if security vulnerabilities or insecure dependencies are detected, preventing them from reaching later stages.

## Language

`python`

## How to Run

Save the code as `devsecops_pipeline.py`.
Run from your terminal: `python devsecops_pipeline.py`
